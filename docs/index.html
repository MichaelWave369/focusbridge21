FocusBridge21

A local-first threshold-training cockpit for original audio, guided practice, safe return, and append-only Signal Ledger receipts.

View on GitHub Read the spec

Not a magic gate button. A practice cockpit.

FocusBridge21 does not promise Focus 21, OBE, therapy, diagnosis, or proprietary signal recreation. It trains capacities: readiness, body-asleep stability, expanded awareness, time-release, stable witness, clean return, and integration.

We do not promise the gate. We train the capacities that make the gate safer, clearer, and more repeatable.

Readiness Gate

Checks mood, sleep, stress, groundedness, environment, headphones, and safety acknowledgement before practice.

Original Audio

Generates transparent binaural WAV sessions locally with gentle ramps and optional noise layers.

Signal Ledger

Writes append-only JSONL receipts so the practitioner can learn personal patterns over time.

Safety + Return

Prominent RETURN NOW grounding protocol, low default intensity, and slow non-flashing visuals.

8-Week Protocol

Capacity-focused training arc from body-asleep practice toward threshold / bridge sessions.

Local First

No cloud required. Runtime data stays on the machine unless intentionally exported.

Run locally

git clone https://github.com/MichaelWave369/focusbridge21.git
cd focusbridge21
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
